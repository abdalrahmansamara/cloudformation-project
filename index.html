Steps to Deploy the CloudFormation Stack

Use the AWS CLI to deploy the static website infrastructure on AWS:

  1. Ensure you have AWS CLI installed and configured with appropriate permissions.
  2. Create an ACM certificate in us-east-1 for your domain using AWS Certificate Manager.
  3. Note your domain name and Route 53 Hosted Zone ID.
    This can be fetched from the AWS Route 53 console.
  4. Download this CloudFormation template file (static-site.yaml) to your local machine.
  5. Run the deploy command (replace parameters below with your actual values):
aws cloudformation deploy \
  --template-file static-site.yaml \
  --stack-name StaticWebsiteStack \
  --capabilities CAPABILITY_NAMED_IAM \
  --parameter-overrides \
    DomainName=yourdomain.com \
    HostedZoneId=Z3P5QSUBK4POTI \
    AcmCertificateArn=arn:aws:acm:us-east-1:123456789012:certificate/abcdefg-1234-5678-abcd-1234abcd5678

Once deployed, the static website will be live on your custom domain with HTTPS enabled!
